# RingMinder

> It actually calls you back.

A voice-based scheduled reminder service for users without smartphones or internet access. Users call a phone number, authenticate with a PIN, enter a date and time via keypad, and leave a voice message. The system calls them back at the scheduled time and plays the message back.

## Documentation

Full product specification: [`docs/PIF.md`](docs/PIF.md)

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (PostgreSQL, Edge Functions, Auth)
- Twilio Programmable Voice
- Netlify (admin UI hosting)
